Question:-What is doctype?

tells the browser to render in standards mode.
It tells the browser which version of HTML the page is written in, so the browser knows how to render it correctly.
without doctype browser goes into Quirks Mode (old Behaviour)

Question:-Which tag is responsible for page responsiveness?
Answer:-The main tag is the viewport meta tag
<meta name="viewport" content="width=device-width, initial-scale=1.0">
width=device-width → match screen width (mobile, tablet, desktop)
initial-scale=1.0 → no zoom by default
Without it,mobile browsers render at desktop width.

Question:-Pseudo-class & element
Answer:-It defines a special state of an element
button:hover {
  background: red;
}

input:focus {
  border: 2px solid blue;
}

li:first-child {
  color: green;
}

It styles a specific part of an element

p::first-letter {
  font-size: 30px;
}

p::first-line {
  color: red;
}

div::before {//content before element
  content: "👉 ";
}

div::after { ///content after element
  content: " ✅";
}

Pseudo-classes target element states like hover or focus, while pseudo-elements style specific parts like first letter or before/after content.
